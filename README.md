## API Server (Development)
현재 App-Hardware-Server 통합 테스트를 위해 외부 노출 서버를 운영 중입니다. 
프론트엔드 및 하드웨어 개발 시 아래 Endpoint를 사용하세요.

- **Base URL**: `https://retorted-rink-reentry.ngrok-free.dev`
- **API Documentation (Swagger)**: [Link](https://retorted-rink-reentry.ngrok-free.dev/swagger-ui/index.html)

> **Note:** 
> 1. 본 서버는 테스트 기간에만 활성화됩니다. 접속이 안 될 경우 팀장에게 서버 기동 요청하시면 됩니다.
> 2. 시연 및 통합 테스트용 주소이므로, 로컬 개발 환경은 기존 `localhost:8080`을 그대로 사용 하시면 됩니다.
